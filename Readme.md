# tf2_conversions
Function set for conversion between geometry_msgs and Eigen. 

# Supported type
* geometry_msgs
  * Point
  * PointStamped
  * Vector3
  * Vector3Stamped
  * Transform
  * TransformStamped
  * Pose
  * PoseStamped
* Eigen
  * Vector3
  * Quaternion
  * Matrix3 (This is only supported in conversion from eigen to msg)
